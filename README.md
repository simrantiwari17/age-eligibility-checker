# Age Eligibility Checker

## Description
A simple web application built using HTML, CSS, and JavaScript that checks whether a user is eligible to vote based on their age.

## Features
- User enters age
- Button click checks eligibility
- Displays result dynamically

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Learned
- DOM manipulation
- Event handling
- Conditional statements
- User input validation (basic)
